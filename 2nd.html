HTML & CSS Basics Summary

HTML Summary

HTML ( HyperText Markup Languge) is used to define our page content, structure and meaning. You don't use it for styling purpose. Use CSS for that instead!

  • HTML uses "elements" to describe (annotate) content
  • HTML elements typically have an opening tag, contene and then a closing tag
  • You can also have void (empty) elements like images
  • You can also configure elements with attributes
  • There's a long list of available elements but you'll gain experience over time, no worries

    • Learn more about all available HTML elements on the MDN HTML elemants reference .

h2>CSS Summary

CSS (Cascading Style Sheets) is used for styling your page content

  • Styling are assigned via property-value pairs
  • You can assign styles via the "style" attribute
  • To avoid can duplication, you typically use global style (via the "style" element) instead
  • Alternatively, you can work with external stylesheet files which yon can "link" to
  • When working with CSS, concepts like "inhertance" "specificity" and the "box model" should be understood.

Learn more about all available CSS properties and values on the MDN CSS property reference.
